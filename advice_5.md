# 📱 Dorm Power 移动端 UI 设计规范（学生端专用版）

适用于：

* Next.js 15 + App Router
* React 19
* Ant Design 5
* PWA 移动端

目标：

> 从“工程界面”升级为“学生真实使用的移动产品”

---

# 一、设计目标定义

## 🎯 产品定位

学生宿舍用电智能控制 App

关键词：

* 清晰
* 安全感
* 易控制
* 快反馈
* 不工程化

---

# 二、整体设计原则

### 1️⃣ 信息优先级

1. 当前是否安全
2. 当前功率
3. 是否异常
4. 是否可控制
5. 历史分析

---

### 2️⃣ 视觉层级原则

| 层级    | 用途                |
| ------- | ------------------- |
| Level 1 | 今日用电 / 当前功率 |
| Level 2 | 插孔状态            |
| Level 3 | 辅助信息            |
| Level 4 | 调试信息            |

---

### 3️⃣ 操作反馈原则

所有控制必须：

* loading 状态
* 成功提示
* 失败提示
* 超时提示

---

# 四、字体规范

## 字体大小体系（移动端）

| 级别     | 用途      | 大小 |
| -------- | --------- | ---- |
| Title XL | 今日用电  | 28px |
| Title L  | 当前功率  | 22px |
| Title M  | 卡片标题  | 18px |
| Body L   | 正文      | 16px |
| Body M   | 辅助      | 14px |
| Caption  | 时间/单位 | 12px |

---

## 字重规范

* 主数值：600
* 标题：600
* 普通文字：400
